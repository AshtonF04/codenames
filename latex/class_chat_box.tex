\doxysubsection{Detailed Description}
A widget for the chat feature in the game. 

This class contains a QText\+Edit for displaying chat messages, a QLine\+Edit for inputting messages, and a QPush\+Button to send messages. It allows players to communicate with each other during the game. It also includes functionality to display system messages and player messages with different styles based on operative guesses and spymaster hints for each team. \begin{DoxyAuthor}{Author}
