\doxysubsection{Detailed Description}
A widget that implements the pre-\/game lobby for multiplayer games. 

The \doxylink{classMultiPregame}{Multi\+Pregame} class manages the pre-\/game setup phase where players join the lobby, select their roles, and prepare for the game to start. It handles both host and client functionality, manages player connections, role assignments, and transitions to the game.
