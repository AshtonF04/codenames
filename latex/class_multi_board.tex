\doxysubsection{Detailed Description}
A widget that implements the multiplayer game board for a team-\/based word guessing game. 

The \doxylink{class_multi_board}{Multi\+Board} class manages the game state, UI, and network communications for a multiplayer word-\/guessing game. It handles player turns, card reveals, scoring, and game progression. The game involves two teams (red and blue) with spymasters giving hints and operators making guesses.
