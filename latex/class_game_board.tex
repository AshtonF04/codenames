\doxysubsection{Detailed Description}
A class representing the game board for the Spy Master game. 

The \doxylink{class_game_board}{Game\+Board} class is responsible for displaying the game board and handling user interactions. It includes methods for loading words from a file, generating the game grid, setting up the UI, card clicks, card reveals and turns, and handling game end conditions. The game board also includes a stacked layout for transitions between screens. Codenames is a game which involves two teams (red and blue) with spymasters giving hints and operators making guesses.
