\doxysubsection{Detailed Description}
A widget that implements the main multiplayer lobby for creating and joining game rooms. 

The \doxylink{classMultiMain}{Multi\+Main} class provides the interface for players to either create a new game room as a host or join an existing game room as a client. It manages Web\+Socket connections for multiplayer functionality and handles the transition to the pre-\/game setup screen.
