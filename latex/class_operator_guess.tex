\doxysubsection{Detailed Description}
A widget that provides the interface for operators to submit guesses during gameplay. 

The \doxylink{class_operator_guess}{Operator\+Guess} class provides a simple UI for team operators to submit their guesses during their turn. It consists of a button that the operator can click to indicate they have made a guess on the game board.
